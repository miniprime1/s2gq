# s2gq: semicolon to the greek quesion mark
 - Replace Semicolon(;) to the Greek Question Mark(Í¾)

<br>

 - [![Tweet](https://img.shields.io/badge/Tweet-1DA1F2?style=flat&logo=twitter&logoColor=white)](https://twitter.com/benbjohnson/status/533848879423578112)

## Usage
```
python s2gq.py <path>
```
<br>

## Example
 - Convert one file.
```
python s2gq.py hello.c
```
 - Convert many files.
```
python s2gq.py hello.c mkdir.c calc.c
```
